# thisiscrap

## Starting the server

```bash
git clone https://github.com/bobby-brennan/thisiscrap.git && cd thisiscrap
npm install
bower install
cd node_modules/strapping && bower install && cd ../..
export DEVELOPMENT=true
node server.js
```
